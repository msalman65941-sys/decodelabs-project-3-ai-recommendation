## 📖 Project Description

This project is a simple rule-based AI Recommendation System developed in Python. It recommends items based on the user's selected category and preference.

The system demonstrates the basic concept of recommendation logic using conditional statements (`if`, `elif`, and `else`) and user input.

---

## ✨ Features

- User-friendly menu
- Three categories:
  - Movies
  - Books
  - Games
- Three types for each category
- Three recommendations for every type
- Input validation
- Simple recommendation logic
